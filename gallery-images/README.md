# Gallery Images

Add photos to these folders:

- `gallery-images/forest-views`
- `gallery-images/architecture`
- `gallery-images/lotus-pond`
- `gallery-images/interiors`
- `gallery-images/waterfall`
- `gallery-images/night-skies`
- `gallery-images/trekking-trail`
- `gallery-images/kitchen`
- `gallery-images/veranda-lawn`

## File naming

Use numeric names so the gallery page can auto-detect files:

- `1.jpg`, `2.jpg`, `3.jpg` ...
- or `1.jpeg`, `1.png`, `1.webp`
- max auto-detected: `30` images per category

After adding images, refresh `gallery-category.html` in the browser.
